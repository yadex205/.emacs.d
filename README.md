Proximacs
=========

Another Emacs configurations.


Setup
-----

### Requirements

* Emacs >= 27

### Recommendations

* Git >= 2.1
* [The Silver Searcher](https://github.com/ggreer/the_silver_searcher)

### Installation

```bash
git clone https://github.com/yadex205/proximacs.git ${HOME}/.emacs.d
```
